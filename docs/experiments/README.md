# Experiments Index

Status: active guidance
Last updated: 2026-05-28

The experiment folders are evidence vaults. They are intentionally large because
they preserve run provenance, failed arms, configs, metrics, and caveats.

Do not infer active guidance from file recency or filename alone. Start with:

- `../current_research_program.md`
- `../component_ceiling_registry.md`
- `gan/README.md`
- `exect/README.md`
- `synthesis/README.md`

## Folder Roles

- `gan/` - Gan S0 frequency experiments and decomposition evidence.
- `exect/` - ExECT field-family, schema-ladder, and component evidence.
- `synthesis/` - cross-experiment, paper, registry, and holdout synthesis.
- `cursor_sdk_drafts/` - review-only drafts; not primary evidence unless promoted.

## Status Rule

An experiment note is historical evidence unless a current index labels it as
`promoted baseline`, `operational default`, `mechanism open`, `rejected arm`,
`diagnostic only`, `paper evidence`, `blocked`, or `superseded`.
